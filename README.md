Tätä autoa saa jakaa eteenpäin kuhan mainitset tekijän. Tätä autoa ei saa myydä eteenpäin missään nimessä!

Asennus Ohjeet :
1.Tarvit tämän käyttöön ELS, lataus: https://github.com/MrDaGree/ELS-FiveM
2.Purkaa tiedosto tähän tarvit WinRAR tai 7zip. Nimeä kansio -> ELS, siirrä ELS resources kansioon, lisää start ELS server.cfg:hen.
3.police.xml tiedosto siirretään vcf kansioon. 
4.Siirrä SuomiTulliAuto resoureces kansioon, lisää start SuomiTulliAuto server.cfg:hen.
